# Visual Studio 

Find resources for Microsoft Visual Studio below

# Downloads

> [!CAUTION]
> Never download software from unofficial sources. For Visual Studio, it is safe to download from aka.ms (Microsoft's
> [official URL shortener](https://en.wikipedia.org/wiki/.ms)) and microsoft.com. **Any other host should be 
> treated as unsafe.**

## Professional/Enterprise

If you have an MSDN license or otherwise have a paid license for Visual Studio, you can download older versions of 
the Professional/Enterprise editions in the All Downloads section of VisualStudio.com.

- https://my.visualstudio.com/Downloads?q=Visual%20Studio
